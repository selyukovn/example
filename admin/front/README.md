
Разработка полноценного FE-приложения (например, SPA + SSR) выходит за рамки данного проекта (по крайней мере сейчас).

Однако, пользовательский интерфейс все же необходим для наглядности.

> Данный компонент является **имитацией FE-приложения**!

Поэтому правила и подходы, распространяемые в рамках проекта на BE-компоненты, здесь не применяются в полной мере. 
Например, нет сбора и экспорта метрик и логов для мониторинга, статические файлы отдаются без кеширования и т.д.
