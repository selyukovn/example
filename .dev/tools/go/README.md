
Для генерации кода из `OpenApi` спецификации используется 
[`github.com/oapi-codegen`](https://github.com/oapi-codegen/oapi-codegen/).

Особенности:

- `json`-ответ должен содержать хотя бы какое-то поле, т.е. не быть пустым, 
  иначе сгенерированный код будет содержать ошибку.
  Очевидно, бага в генераторе.

- нельзя менять сообщение статуса ответа (я не нашел возможности)
